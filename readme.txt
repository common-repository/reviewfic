== Description ==

Reviewfic is the ultimate plugin for showcasing testimonials, customer reviews, or quotes on your WordPress site. It allows you to easily display reviews in various formats across posts, pages, custom templates, widgets, and more. With its intuitive Shortcode Generator and live preview feature, creating stunning and unique testimonials has never been easier. The plugin offers fully responsive layouts and highly customizable styling options to match the look and feel of your website.

<h4>Major features</h4>

<ul>
<li>Show Client Reviews on any pages/posts/custom post types</li>
<li>A Shortcode generator to generate shortcode</li>
</ul>

<h4>How to Install</h4>
You can use our built-in shortcode generator or simply paste the following shortcode anywhere on your site:

<pre>[reviewfic category="all" columns="" max_items="Unlimited"]</pre>

<ul>
<li>Replace "all" in the category parameter with your preferred category to display specific testimonials.</li>
<li>For the columns parameter, enter 1, 2, 3, or 4 to define the number of columns to display.</li>
<li>In the max_items parameter, input any number (e.g., 1, 2, 3…) to limit the number of testimonials displayed.</li>
</ul>

== Installation ==

1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation.
2. Active the plugin from the WordPress Plugins administration page OR, Go to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'Reviewfic'.
3. Install and activate the plugin.
4. Add the shortcode shown above on any pages/posts/custom post types.
